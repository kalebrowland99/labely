```json
  "comment" : "Number of items",
  "localizations" : {
    "en" : {
      "variations" : {
        "plural" : {
          "one" : {

              "value" : "%lld item"
            }
          },
          "other" : {

              "state" : "translated",
              "value" : "%lld items"
            }
          }
        }
      }
    },

      "variations" : {
        "plural" : {
          "one" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld artículo"

          },
          "other" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld artículos"
            }
          }
        }
      }
    },
    "ru" : {
      "variations" : {
        "plural" : {
          "one" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld элемент"
            }
          },
          "few" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld элемента"
            }
          },
          "many" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld элементов"
            }
          },
          "other" : {
            "stringUnit" : {
              "state" : "translated",
              "value" : "%lld элемента"
            }

```

**Usage in code:**
```swift
// Automatically uses correct plural form for each language
Text("items_count", comment: "Item count")
```

## 📋 Quick Copy-Paste Format

To add any of these to your `Localizable.xcstrings`, just copy the entire JSON block (including the key name and outer braces) and paste it into the `"strings"` section of your file, making sure to add a comma between entries.

---

## 🎯 Priority Strings to Add First

Start with these based on your app's main features:
1. Navigation items (home, invoices, clients, settings)
2. Core actions (save, cancel, delete, edit)
3. Status labels (paid, unpaid, draft)
4. Invoice fields (total, date, client)
5. Messages (success, error, loading)

---

**Need more?** Just follow the same pattern and use Google Translate or a professional translator for Spanish/Russian!
